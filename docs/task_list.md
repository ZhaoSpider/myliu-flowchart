## 当前存在的技术问题

### 🔴 严重问题

| 问题 | 影响 | 解决方案 |
|------|------|----------|
| AntV X6 未集成 | 流程图核心功能无法开展 | 优先集成 X6 引擎 |
| Controller 层为空 | 无业务 API 接口 | 开发后端 API |
| 数据库表未创建 | 只有实体类，无业务表 | 编写初始化脚本 |

### 🟡 中等问题

| 问题 | 影响 | 解决方案 |
|------|------|----------|
| 前端组件目录为空 | 编辑器功能只有框架 | 实现画布组件 |
| API 接口未对接 | 前后端联调未开始 | 对接接口 |

### 🟢 小问题

| 问题 | 影响 | 解决方案 |
|------|------|----------|
| Maven 本地未安装 | 开发依赖 Docker Maven | 可选安装本地 Maven |

## 开发计划建议

### 第一阶段（1-2 周）：核心功能
1. 集成 AntV X6 引擎
2. 实现基础节点和连线
3. 开发后端基础 API
4. 数据库表初始化

### 第二阶段（2-3 周）：功能完善
1. 文件 CRUD 完整功能
2. 导出功能实现
3. 用户注册/认证完善
4. 前后端联调

### 第三阶段（1-2 周）：优化扩展
1. 文件版本管理
2. 性能优化
3. UI/UX 优化
4. 测试和 Bug 修复
