## 当前存在的技术问题

### 🔴 严重问题

| 问题                   | 影响                            | 解决方案                                           |
| ---------------------- | ------------------------------- | -------------------------------------------------- |
| Controller 层为空/缺失 | 无业务 API 接口，前后端无法联调 | 开发后端 API（auth/files/content/versions/export） |
| 前端保存未对接         | 编辑器“保存”目前仅本地 JSON     | 对接后端保存与版本接口                             |

### 🟡 中等问题

| 问题                      | 影响                                | 解决方案                    |
| ------------------------- | ----------------------------------- | --------------------------- |
| API 接口未对接/后端缺口大 | FilesView/EditorView 依赖接口不可用 | 先补齐后端接口，再做联调    |
| JWT secret 配置规范       | 0.12.x 建议 Base64 且满足长度       | 统一配置方式并补充文档/校验 |

### 🟢 小问题

| 问题                     | 影响                | 解决方案                                          |
| ------------------------ | ------------------- | ------------------------------------------------- |
| 本地 Maven/Java 环境差异 | 编译/启动方式不一致 | 统一使用 Docker Compose 或补充本地环境说明        |
| docker-compose 明文密码  | 有泄露风险          | 用 `.env`/环境变量或本地覆盖配置（开发/生产分离） |
